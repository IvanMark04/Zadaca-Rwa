Pristupite resursu s vašeg računala putem preglednika (po vašem izboru):

http://info.cern.ch/hypertext/WWW/TheProject.html

te odgovorite na pitanja (svaki odgovor pokrijepiti analizom ili dijelom HTTP poruke koji se odnosi ili nam daje taj odgovor):


a)	Pokreće li vaš preglednik HTTP verziju 1.0 ili 1.1? 
        
        Preglednik koristi HTTP/1.1, što je vidljivo iz HTTP odgovora:

        HTTP/1.1 304 Not Modified


b)	Koja je verzija HTTP-a poslužitelja pokrenuta kad pristupite npr sa sum.ba?

        Poslužitelj na stranici sum.ba koristi HTTP/1.1 protokol, što je vidljivo iz prve linije HTTP odgovora:

        HTTP/1.1 304 Not Modified
      

c)	Koje jezike (ako postoje) vaš preglednik pokazuje da može prihvatiti na poslužitelju?

        Accept-Language: en-US,en;q=0.9,hr-HR;q=0.8,hr;q=0.7

        Preglednik prihvaća engleski (US), engleski i hrvatski jezik.

d)	Koja je IP adresa vašeg računala? A koja je IP adresa sum.ba?

        192.168.2.212

        Dobivena je pomoću naredbe ipconfig, gdje je prikazana vrijednost IPv4 Address.

        IP adresa poslužitelja sum.ba je:

        212.39.111.33

        Dobivena je pomoću DNS upita (nslookup sum.ba) odnosno iz mrežnih podataka prilikom pristupa stranici.


e)	Kada je HTML datoteka koju dohvaćate zadnji put modificirana na poslužitelju?

        Thu, 03 Dec 1992 08:37:20 GMT

        U preglednik se vraća 2217 bajtova sadržaja, što je vidljivo iz HTTP zaglavlja:

        Content-Length: 2217


g)	Koji softver/web poslužitelj  poslužje web stranicu na server,navedite odakle vidite?

2.2. Pristup različitim resursima

Pristupite sljedećim resursima te za svaki resurs napišite HTTP odgovor koji dobijete (bez tijela), te za svaki resurs odgovorite:

a)	kojem MIME formatu resursa se radi:
          MIME format je application/pdf.
b)	kolika je veličina resursa (u byte-ima)
          Veličina resursa nije navedena pomoću Content-Length jer se prijenos vrši pomoću:
          Transfer-Encoding: chunked
          pa se veličina ne može očitati iz HTTP zaglavlja.
c)	koja je verzija HTTP poslužitelja
          Verzija HTTP poslužitelja je HTTP/1.1.
